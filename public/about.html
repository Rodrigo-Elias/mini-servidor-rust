A Eficiência do Rust

Este mini-servidor foi criado para demonstrar habilidades em programação de sistemas e rede.

  • **Assincronicidade:** Utiliza o *runtime* Tokio para lidar com múltiplas requisições simultaneamente.
  • **Segurança:** A segurança de memória do Rust previne falhas comuns em servidores web.
  • **Simplicidade:** Implementação manual de um servidor HTTP para fins didáticos.
